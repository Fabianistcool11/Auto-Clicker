# Epic Auto Clicker 🖱️💥

**Description / Beschreibung:**  
**English:** Epic Auto Clicker for Windows! Set custom intervals in ms, sec, min, or hours. Start with F6, stop with F7. Simple GUI, reliable clicks, perfect for gaming or automation. Adjustable and easy to use.  


**Deutsch:** Epic Auto-Clicker für Windows! Intervalle in ms, Sek, Min oder Std einstellbar. Start mit F6, Stopp mit F7. Einfaches GUI, zuverlässiges Klicken, ideal für Spiele oder Automatisierung. Anpassbar und benutzerfreundlich.

## Features / Funktionen
- Set custom click intervals: ms / sec / min / hours  
- Start / Stop with **F6 / F7**  
- Simple and clean GUI  
- Keyboard support  
- Easy to use & reliable  
- Lightweight Python app  

## Usage / Nutzung

1. Install Auto-Clicker.exe  
2. Run `Auto-Clicker.exe`  
3. Set interval and unit (ms/Sek/Min/Std)  
4. Press **F6** to start clicking, **F7** to stop  

## Notes / Hinweise
- Keep the app window open for hotkeys to work  
- Works on Windows only  
- Customize interval for safe clicking
